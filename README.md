# Social Network

## What is it?
I've created this project some two years ago to use as a lab to learn AngularJS, Bootstrap, front-end development in general + Spring Boot in the backend.

This project was originally divided into multiple repositories in Bitbucket and I merged them here on Github. 

As with most of my project, I was developing this more as a 'learning ground' than an actual project with a clear goal. So the application is not complete and probably it won't ever be.
